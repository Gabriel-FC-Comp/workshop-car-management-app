# Workshop Car Management App

![Rust](https://img.shields.io/badge/Rust-CE422B?style=flat&logo=rust&logoColor=white)
![Tauri](https://img.shields.io/badge/Tauri-FFC413?style=flat&logo=tauri&logoColor=black)

This repository is a project developed to help manage incoming car services in an automotive workshop, aiming to help keep track of the current cars being worked on, the ones waiting, and the ones scheduled for later days, to improve the workflow and help enforce better exchange of information about the cars between mechanics and counter staff.
